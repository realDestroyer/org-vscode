# Org-vscode How-To

> Organize your thoughts and tasks into hierarchical lists.  
> - Create items using `*`. The number of `*` determines the heading level.  
> - Mark tasks as `TODO` or `DONE`.  
> - Fold lists with `Tab`.  
> - Increment or decrement headings using `Alt + Left/Right`.
